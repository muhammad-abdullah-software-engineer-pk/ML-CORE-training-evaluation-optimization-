# 🤖 ML-CORE: Training, Evaluation & Optimization

A hands-on Machine Learning training program covering core concepts from bias/variance to ensemble learning, built on the Titanic dataset using Python & Scikit-Learn.

---

## 📚 Modules

| # | Topic |
|---|-------|
| 1 | Bias vs Variance |
| 2 | Overfitting & Underfitting |
| 3 | Feature Engineering |
| 4 | Cross-Validation & Metrics |
| 5 | Core Algorithms (LR, DT, RF, XGBoost, SVM) |
| 6 | Hyperparameter Tuning (GridSearch / RandomizedSearch) |
| 7 | Ensemble Learning (Bagging, Boosting, Stacking) |

---

## 🛠️ Stack

`Python` · `scikit-learn` · `pandas` · `numpy` · `xgboost` · `seaborn` · `matplotlib`

---

## ✅ Key Takeaways

- Good features beat fancy algorithms
- Use F1-score over accuracy on imbalanced data
- Pipelines prevent data leakage and ensure reproducibility
- RandomizedSearch is faster than GridSearch with comparable results
- Ensemble methods consistently outperform single models
